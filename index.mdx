## How It Works

<Steps>
  <Step title="Submit Your Repository">
    Send a POST request with your GitHub repository URL to the upload service
  </Step>
  <Step title="Clone and Upload">
    The platform clones your repository and uploads all files to S3 storage
  </Step>
  <Step title="Queue the Build">
    Your deployment is added to the Redis build queue for processing
  </Step>
  <Step title="Docker Build">
    A Docker container builds your project, handling dependencies and bundling automatically
  </Step>
  <Step title="Deploy to S3">
    The built assets are uploaded to S3 and made available via a unique deployment ID
  </Step>
</Steps>

## Quick Start

Get your first deployment running in under 5 minutes.

<CodeGroup>

```bash cURL
curl -X POST http://localhost:3000/get/url \
  -H "Content-Type: application/json" \
  -d '{"url": "https://github.com/your-username/your-frontend-repo"}'
```

```javascript JavaScript/Node.js
const response = await fetch('http://localhost:3000/get/url', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    url: 'https://github.com/your-username/your-frontend-repo'
  })
});

const { id } = await response.json();
console.log('Deployment ID:', id);
```

```python Python
import requests

response = requests.post(
    'http://localhost:3000/get/url',
    json={'url': 'https://github.com/your-username/your-frontend-repo'}
)

data = response.json()
print(f"Deployment ID: {data['id']}")
```
